2025년 1월 9일 목요일에 열린 .NET Conf 2025 x Seoul의 발표 자료를 이곳에서 찾아보실 수 있습니다. 링크가 걸려있지 않은 항목은 준비 중이거나 발표자 사정으로 공개가 어려운 세션입니다.
